LineBot聊天機器人

for event in events:
if isinstance(event, MessageEvent):
if isinstance(event.message, TextMessage):
mtext = event.message.text
if mtext == ‘那你覺得我午餐該吃什麼’:
func.sendText(event)
elif mtext == ‘@傳送圖片’:
func.sendImage(event)
elif mtext == ‘@傳送貼圖’:
func.sendStick(event)
elif mtext == ‘你午餐吃什麼呢’:
func.sendMulti(event)
elif mtext == ‘附近有什麼餐廳嗎’:
func.sendPosition(event)
if mtext == ‘有推薦的餐點嗎’:
func.sendQuickreply(event)
return HttpResponse()
else:
return HttpResponseBadRequest()

心得:
學習了幾個月的程式後,終於設計出一個作品了,在設計程式的過程中,遇到了一些困難(import module時發生錯誤、在撰寫程式碼時語法不正確等),藉由網路資源及書籍,最終找尋出合適的方法,經過一兩個禮拜的努力後,終於完成了現在的聊天機器人.

在撰寫程式碼時,有時會感到疲倦甚至是煩躁,但只要換個角度想,咬緊牙度過了這關後,自己又離目標更近了一點,也就更有動力的往下繼續努力.

最重要的是,經過這次的實作,了解到學習程式和設計程式有很大的不同,一個需要的是記憶,一個需要的是思考,今後也要多多練習實作培養出應有的計算思維.
